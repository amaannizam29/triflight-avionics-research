# TriFlight Minimum Evidence Package

Before claiming completed engineering implementation, collect the following evidence.

| Evidence Item | Purpose | Status |
|---|---|---|
| Clean wiring diagram | Shows system architecture | Complete (design) |
| Component table with part numbers and cost | Shows justified selection | Complete (design) |
| Firmware parameter export (.param) | Proves FC configuration | Pending hardware |
| MAVLink CSV log from SITL or real FC | Proves telemetry logger works | Code complete; SITL pending |
| Bench voltage/current measurements | Proves electrical validation | Pending hardware |
| GPS/compass calibration screenshots | Proves sensor setup | Pending hardware |
| Signed preflight checklist | Proves safe procedure | Pending hardware |
| Flight log plots | Proves experimental results | Pending hardware |
| Post-flight inspection photos | Proves airworthiness check | Pending hardware |
